# tic-tac-toe

An interactive tic-tac-toe game.

## Setup
Fork/Clone the repo onto your local system
```
npm install
```
```
npm run build
```
```
npm start
```

## Technologies 
* React 
* HTML/CSS
* Webpack
* chai

## Testing
To run the tests type the following command in your terminal

```
npm test
```
